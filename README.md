# EcoImpact-Frontend

# Installation
Clone the repository:
   git clone https://github.com/moizbinkhalil/EcoImpact-frontend.git
Navigate into the folder:
   cd EcoImpact-frontend
Install dependencies:
   npm install
Start the development server:
   npm start
The application will open in your browser at http://localhost:3000

# Features
Interactive Home Page - Animated hero section with multilingual text transitions (English, Urdu, Chinese) and scrolling indicator animations
Policy Simulation - Configure and simulate climate policies with customizable parameters including duration, carbon price rate, and country selection
Policy Comparison - Compare two different climate policies side-by-side to analyze their relative impacts
User Authentication - Sign up and login system with form validation, ready for backend integration

# Project Structure
src/App.js - Main React component containing all pages and logic
src/index.css - All styling and animations
src/images/ - Image assets (logo, Earth, cyber Earth, etc.)
public/index.html - Main HTML file
